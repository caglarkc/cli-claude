# CLI Claude

CLI Claude is a lightweight command-line project space for Claude-oriented tooling and experiments.

## Project Scope

- `src/`: source code and CLI-related implementation
- `SRC_ENVENTORY_FULL_TR.txt`: Turkish source inventory/reference notes

## Usage

This repository is currently organized as a development sandbox.  
Start by reviewing `src/` and adapting commands/scripts for your environment.

## Notes

- Keep secrets out of the repository (`.env`, keys, tokens).
- Prefer documenting command flows and setup expectations directly in `src/` as the project grows.
