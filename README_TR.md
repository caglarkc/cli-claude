# CLI Claude

CLI Claude, Claude odaklı araçlar ve deneyler için hafif bir komut satırı proje alanıdır.

## Proje Kapsamı

- `src/`: kaynak kod ve CLI uygulama katmanı
- `SRC_ENVENTORY_FULL_TR.txt`: Türkçe kaynak envanteri/notları

## Kullanım

Bu depo şu an geliştirme sandbox yapısında düzenlenmiştir.  
Önce `src/` içeriğini inceleyip komut/script akışlarını kendi ortamınıza göre uyarlayın.

## Notlar

- Gizli bilgileri repoya koymayın (`.env`, key, token).
- Proje büyüdükçe kurulum ve komut akışlarını `src/` içinde dokümante edin.
